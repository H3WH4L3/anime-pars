# Мой проект на Python

## Описание
Это скрипт для парсинга аниме через Kodik.

## Установка
1. Клонировать репозиторий.
2. Создать виртуальное окружение:
   ```bash
   python -m venv venv
   ```
3. Активировать окружение:
   - Windows:
     ```bash
     .\venv\Scripts\activate
     ```
   - Linux/macOS:
     ```bash
     source venv/bin/activate
     ```
4. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование
На данном этапе всё использование происходит через прямой запуск скрипта:
```bash
python anime.py
```
